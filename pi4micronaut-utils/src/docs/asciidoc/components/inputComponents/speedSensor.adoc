===== YAML Configuration
The YAML configuration for the Speed Sensor is as follows:

[source, yaml]
----
  speed-sensor:
    name: Speed Sensor           # <1>
    address: 22                  # <2>
    pulses-per-revolution: 20    # <3>
    provider: pigpio-digital-input # <4>
    debounce: 500                # <5>
----
So, the input of the tilt switch is connected to GPIO 17.
